#

## 安装 pm2-windows-service

```
pnpm add pm2-windows-service -g
```

## 创建服务

```
pm2-service-install -n serviceName
```

## 启动应用

```
pm2 start app.js | ecosystem.config
```

## 设置 PM2_HOME 环境变量

::: warning
如果不设置，默认的 pm2 save 的信息会存在用户的目录下，重启服务可能没有权限读取
:::
在 GUI 中添加环境变量指定路径

## 保存 pm2 配置信息

```
pm2 save
```

## 卸载服务

```
pm2-service-uninstall -n serviceName
```
